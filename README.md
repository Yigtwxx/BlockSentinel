# BlockSentinel
Bu platform, EV şarj istasyonlarının hayati CAN-Bus iletişim altyapısındaki siber açıkları tespit, izleme ve önleme amacıyla geliştirilmiştir. Sistem, CAN hattındaki veri trafiğini gerçek zamanlı analiz eder. Anormal davranışları anında yakalar ve tüm bulguları, değiştirilemezlik sağlayan blok zinciri ağına kaydeder.
